# 🏦 Bank Management System (Python)

## 📌 Project Overview
This is a **Bank Management System** developed in Python. The project is a console-based application that allows users to perform basic banking operations such as:

- Create a new bank account
- Login using account details
- Deposit money
- Withdraw money
- Check account balance
- Store customer information in a JSON file

The project uses **Object-Oriented Programming (OOP)** concepts and **JSON** for data storage.

---

## 🛠️ Technologies Used
- Python 3
- JSON
- OOP (Classes and Objects)

---

## 📂 Project Structure

```
bank_management_project/

├── main.py          # Main application file
```
